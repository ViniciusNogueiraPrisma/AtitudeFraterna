O novo site da Atitude Fraterna foi desenvolvido para oferecer acessibilidade e melhor navegabilidade aos seus visitantes.

O conteúdo do site foi adaptado com o intuito de se tornar mais claro e prático, com nova diagramação, em que o layout permite a inserção das regras de acessibilidade web.
O site foi desenvolvido acompanhando o W3C - World Wide Web Consortium, a comunidade internacional que desenvolve os padrões da web e a acessibilidade comprovada através de testes com os programas mais utilizados para a acessibilidade de pessoas com deficiência visual e auditiva, com os navegadores Firefox, Internet Explorer, Chrome e Safari.
Caso o usuário encontre qualquer dificuldade ou possua sugestões para aperfeiçoar a qualidade e acessibilidade do site, clique aqui.

Navegação pelo teclado

Os conteúdos de todas as páginas foram criados para facilitar a navegação do usuário através das teclas de atalho.

  • Home = Alt + 1
  • Busca = Alt + 2
  • Pular Conteudo = Alt + 3
  • Aumentar Zoom = Alt + 4
  • Diminuir Zoom = Alt + 5
  • Português = Alt + 7
  • Inglês = Alt + 8
  • Topo = Alt + 9

Firefox:

  • Alt + Shift + número

Opera:

  • Shift + Escape + número

Safari e OmniWeb:

  • Ctrl + número

É possível também navegar entre os links de uma determinada página usando a tecla TAB.

Tradutores para Libras

Vlibras: Clique aqui para acessar a página deste plugin, que pode ser usado em diversos devices e plataformas. O Vlibras aciona um recurso que traduz informações em Português como textos, áudios e vídeos para a língua brasileira de sinais (LIBRAS) e torna o conteúdo do site Atitude Fraterna acessível para pessoas com deficiência auditiva.

Este site tem melhor acessibilidade quando acessado nas versões mais atualizadas do seu navegador. Utilize sempre a versão mais recente de seu software.
